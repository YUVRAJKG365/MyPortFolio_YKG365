Self-Learning Facial Emotion Recognition System

Python OpenCV TensorFlow/Keras Streamlit

Developed a production-ready ML application that captures camera input, classifies facial expressions in real time with 89% initial accuracy, and employs a self-learning neural network that retrains on newly collected data to continuously improve prediction performance. The system uses a convolutional neural network (CNN) architecture with transfer learning techniques to achieve high accuracy with relatively small training datasets. The frontend interface was built with Streamlit for easy deployment and user interaction.

Python Streamlit Gemini API Plotly

Built an interactive dashboard that automates data ingestion, cleansing, and feature engineering. Automated Hospital Management System

Python Streamlit MySQL OpenAI API

Engineered a comprehensive full-stack solution for hospital management with modules for bed allocation, inventory tracking, ambulance dispatch, appointment scheduling, and discharge processing.
